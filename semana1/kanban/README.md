# Kanban Board

A single-board kanban project management app.

## Setup

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Testing

```bash
npm test              # unit tests
npx playwright test   # e2e tests
```

## Build

```bash
npm run build
npm start
```
